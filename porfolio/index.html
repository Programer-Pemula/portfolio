About Me

Halo, saya Tegar, seorang mahasiswa Universitas Tangerang Raya (UNTARA) semester 5 yang memiliki passion di bidang web development. Selain kuliah, saya juga bekerja sebagai buruh di PT Hanaqua Industrial Corpindo. Kesibukan sehari-hari antara bekerja dan kuliah tidak menghalangi saya untuk terus belajar dan mengembangkan keterampilan, terutama di dunia pemrograman.

Rutinitas saya cukup padat, mulai dari bekerja, kuliah, hingga menyisihkan waktu di malam hari untuk ngoding. Meskipun lelah setelah aktivitas seharian, saya selalu berusaha meluangkan beberapa jam untuk menulis kode. Bagi saya, ngoding bukan sekadar hobi, melainkan jalan untuk mengasah logika, kreativitas, serta membuka peluang di masa depan.

Saat ini saya menguasai HTML hingga 80%, CSS 75%, dan JavaScript sekitar 50%. Dengan pondasi ini, saya ingin terus mengembangkan diri agar bisa menjadi seorang frontend developer yang andal dan profesional.
